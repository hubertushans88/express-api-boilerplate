# Express-API (Personal) Boilerplate
Simple Express JS API with MongoDB 

## Using
- ExpressJs
- MongoDB 
- [Mongoose](https://mongoosejs.com/) as ODM
- [Joi](https://joi.dev/api/) as validator
- BCRYPT



## Folder Structure
```
|- src
  |- config
  |- controllers
  |- middlewares
  |- models
  |- routes
  |- utils
  |- validations
```
