# Express-API (Personal Boilerplate)

## Using
- ExpressJs
- MongoDB 
- [Mongoose](https://mongoosejs.com/) as ODM
- [Joi](https://joi.dev/api/) as validator
- BCRYPT

```
  |- controllers
  |- middlewares
  |- models
  |- routes
  |- utils
  |- validations
```
